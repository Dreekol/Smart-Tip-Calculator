# Smart Tip Calculator

A lightweight Next.js + Tailwind CSS tip calculator with:
- quick-tip presets
- a custom tip slider
- bill splitting
- round-up and round-down controls
- a dark, mobile-friendly layout

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal. If port 3000 is busy, Next.js will try the next available port automatically.

## GitHub status

This repository is already connected to the GitHub remote and has been pushed to the `main` branch.

## Deploy on Vercel

Deployment should be the final step after the repo is ready.

1. Open https://vercel.com/new
2. Import the GitHub repository
3. Keep the default project settings
4. Click **Deploy**

Vercel will detect the Next.js app automatically and publish it for you.
